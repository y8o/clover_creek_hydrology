Clover Creek Hydrology Modeling

Raster-based GIS project using ArcGIS Desktop to simulate water flow, stream networks, and watershed sub-basins for the Clover Creek watershed in Arizona.

Tools & Techniques

ArcGIS DEM Analysis Flow Direction Stream Order Watershed Tool Raster Calculator

Key Findings

  • Majority of flow is toward the southeast.
  • Identified 1,849 stream cells (accumulation ≥ 500).
  • 5th-order streams are the highest in the network.
  • 17 sub-basins delineated from the stream links.
  • Largest sub-basin contains 6,402 pixels.
